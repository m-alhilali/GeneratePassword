# 🔐 Password Generator

A simple C# application to generate random and secure passwords with a graphical interface using **Windows Forms**.

---

## ✨ Features
- Choose character types to include in the password:
  - Numbers (0–9)
  - Uppercase letters (A–Z)
  - Lowercase letters (a–z)
  - Special characters (!, @, #, ...)
  - Mixed random option for all types
- Set password length using a **TrackBar**.
- Copy the generated password with one click.
- Reset all settings easily.

---

## 📂 Project Structure
- **Form1.cs**: Contains the logic for password generation and UI handling.
- **Reset()**: Resets all options.
- **GeneratePassword()**: Creates a new password based on selected options
